Detailed Description

The Simulation class. This sets up the GUI and the drawing environment.

Simulation::Simulation ( int  argc,
char *  argv[],
int  width,
int  height

The constructor for the Simulation class. It does all setup for the display. In particular, the initial values are set for the robot, obstacles, target, and walls. It takes four parameters: the number of console arguments, their values, and the width and height of the screen in pixels.

Parameters
